# Agriculture Crop Production Prediction

## Overview
This project predicts future agriculture crop production in India using Machine Learning.

## Dataset
The dataset contains historical year-wise crop production values.

## Model Used
Linear Regression

## Prediction
The model predicts crop production for future years:
- 2025
- 2026
- 2027

## Tools
Python, Pandas, Scikit-learn, Jupyter Notebook

## Output
The output is an array of predicted production values generated by the trained model.
